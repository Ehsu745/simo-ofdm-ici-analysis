# A 階段筆記：時變通道與 ICI 複現

## 目標
在修正後的 SIMO-OFDM v2 上引入時變通道（都卜勒效應），複現高移動性場景下
OFDM 子載波正交性被破壞、產生 ICI（Inter-Carrier Interference，載波間干擾）的現象。

## 關鍵參數
**歸一化都卜勒頻率 `fdTs` = 最大都卜勒頻率 / 子載波間距**
值越大代表通道在一個 OFDM 符號內變化越快，正交性破壞越嚴重、ICI 越強。
參考文獻：實際行動 WiMAX 環境 fdTs 約 0.2 時 ICI 已嚴重到需要進階等化器處理。
本模擬掃描範圍 0 ~ 0.25，涵蓋「正交性良好」到「明顯 ICI」。

## 兩種時變通道模型
- **`phase`（簡化模型）**：每個 tap 單一隨機都卜勒分量，各 tap 方向獨立。
  直觀展示「通道在符號內不再是常數 → FFT 不再對角 → ICI」的來源。教學用。
- **`jakes`（業界標準）**：每 tap 由 16 條正弦分量疊加（sum-of-sinusoids），
  自相關呈零階貝索函數 J₀(2π·fd·Δt)，標準 Rayleigh 時變模型。嚴謹驗證用。

## 三個產出
1. **星座圖對比**：fdTs 由 0 增大，星座點由清晰四簇逐漸模糊散開。
2. **子載波洩漏圖**：單一子載波激發，觀察功率洩漏到鄰近子載波（ICI 的直接證據）。
3. **BER / ICI 功率 vs fdTs**：量化傳統單抽頭等化器在都卜勒上升時的失效。

## 驗證結果（NumPy port）
| fdTs | 洩漏比例 (jakes) | BER (jakes) |
|------|-----------------|-------------|
| 0    | 0.0000          | 2.9e-3      |
| 0.05 | 0.0035          | 8.5e-3      |
| 0.1  | 0.038           | 1.9e-2      |
| 0.2  | 0.107           | 5.7e-2      |

洩漏與 BER 均隨都卜勒單調上升、零都卜勒時洩漏為零（正交性完美）→ 物理正確。

## 開發過程中抓到的一個重要錯誤（reflection 素材）
初版時變模型讓**所有 tap 同方向相位旋轉**，這在數學上等於對訊號疊加一個
**固定載波頻偏（CFO）**，而非真正的 ICI。症狀是 BER 暴衝到 ~0.75（接近 QPSK
亂猜上限），整個星座一起旋轉而非模糊散開。

**根因**：混淆了兩個不同的物理現象——
- CFO（載波頻偏）：整體頻率偏移，星座**旋轉**，可由單一相位補償校正。
- ICI（載波間干擾）：符號內通道變化破壞正交性，星座**模糊散開**，無法用單一相位校正。

**修正**：讓每個 tap、每條正弦路徑各自獨立衰落、都卜勒方向不同，
使整體平均頻偏為零，只留下「符號內通道變化」這個真正產生 ICI 的成分。
修正後 BER 呈平滑誤碼地板（3e-3 → 5e-2），物理正確。

> 這個錯誤示範了「程式跑得出數字，但數字背後是錯的物理」——
> 子載波洩漏圖一直是對的，但 BER 曲線被 CFO 污染。
> 透過交叉檢查「星座是旋轉還是散開」才定位出問題。
> 區分 CFO 與 ICI、並用獨立衰落修正，是這階段最有價值的收穫。
